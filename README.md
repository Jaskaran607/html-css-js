# 🚀 Iron Man 3D Armory

![Iron Man Banner](https://i.imgur.com/JZ5X3yA.jpg)

An interactive 3D showcase of Tony Stark's suits with technical specifications and Avengers recruitment portal.

## 🌟 Features

- **3D Model Viewer**: Rotate and zoom the Mark LXXXV suit
- **Suit Gallery**: Explore all MCU suits with hover details
- **Tech Specs**: HUD-style breakdown of armor capabilities
- **S.H.I.E.L.D. Database**: Themed contact form
- **Responsive Design**: Works on mobile & desktop

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **3D Viewer**: [Sketchfab Embed](https://sketchfab.com)
- **Fonts**: Google Fonts (Orbitron, Electrolize)
- **Icons**: Font Awesome
